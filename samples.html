ROBLOX Places:

Ethics Committee Utility

Description: I made this so that people are able to search up user's groups who are not in-game using the groups API. It checks certain departments in a group based on what boxes are checked. It will give a list with the ability to be copied of everyone who is in a hostile group or below a certain rank.

Link: https://www.roblox.com/games/5965939145/Ethics-Committee-Utility

Elevator Worksite

Description: I figured every building needs an elevator, and so why not make one for the fun of it? So that's exactly what I did. I made this elevator completely from scratch, I built and scripted it entirely although I used a couple of royalty free sound effects. It functions extremely well but is still in the works, hopefully with many more neat features to come. But the one particular elevator here features 15 floors with the ability to lock certain floors and a door open & door close button.

Link: https://www.roblox.com/games/5921609320/Elevator-Workplace

 

Guess The Song

Description: I got bored one day and decided to challenge myself to make a game, and so I thought of making a Guess The Song game. My version of the popular game contains several neat features such as: A non-repeating queue of songs (until they are all played at least once), a Win counter with working datastore-service, and of course the method of easily adding in new songs as well as multiple keywords for each song. That way there is no exact guess to get you the win.

Link: https://www.roblox.com/games/4868756537/ALPHA-Guess-The-Song

Message Cipher Center

Description: This game is kind of amazing, it allows you to encode or decode messages with the different ciphers within it. So far there are only two; Backwards Code, and Shifted Code. It's a very complex string manipulation script, but works well.

Link: https://www.roblox.com/games/5593827287/Cipher-Central
